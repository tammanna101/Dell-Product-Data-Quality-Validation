# Dell Product Data Quality & Attribute Validation

## Project Overview
This project focuses on validating Dell product catalog data and identifying data quality issues such as missing attributes, invalid values, and Product Not Found records.

Developed as part of Data Quality and Product Catalog Validation activities for Dell Technologies through eClerx Services.

## Business Problem
Large product catalogs often contain:
- Missing attribute values
- Incorrect product specifications
- Product Not Found records
- Data inconsistencies across source systems

These issues impact reporting accuracy and business decision-making.

## Objectives
- Validate product attributes
- Identify missing values
- Detect Product Not Found records
- Monitor data quality KPIs
- Improve catalog accuracy

## Dataset Summary
- Total Records: 500,231+
- Unique SKUs: 12,000+
- Product Categories: Battery, Adapter, Power, Accessories
- Validation Types: Valid, Missing, Product Not Found
